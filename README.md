

# MedAlert AI: Ensemble Health Risk Prediction System

MedAlert AI is an intelligent healthcare decision-support system designed to identify early risk patterns for major chronic diseases. By combining **Machine Learning (Logistic Regression)** with **Generative AI (Llama 3.1)**, the platform provides users with both quantitative risk assessments and qualitative, actionable health advice.

## 🚀 Key Features
* **Multi-Disease Risk Assessment**: Dedicated modules for Diabetes, Breast Cancer, Heart Disease, Kidney Disease, and Liver Disease.
* **AI-Powered Guidance**: Personalized health recommendations generated by Llama 3.1 (via Groq API) based on specific risk levels.
* **Smart Alert System**: An in-app notification system that triggers health reminders at tiered intervals (1, 2, or 3 hours) depending on the severity of the detected risk.
* **Medical Facility Locator**: Integration with OpenStreetMap (Nominatim and Overpass API) to find nearby hospitals based on live geolocation or a specific place name.
* **Comprehensive History**: A secure user dashboard and history log to track previous assessments and review AI advice.

## 🛠️ Technology Stack
* **Backend**: Python / Flask
* **Frontend**: HTML5, CSS3 (Modern UI with Inter/Poppins fonts), JavaScript (Vanilla)
* **Database**: MongoDB
* **Machine Learning**: Scikit-Learn (Logistic Regression, StandardScaler)
* **Generative AI**: Groq Cloud SDK (Llama 3.1-8b-instant)
* **External APIs**: OpenStreetMap (Nominatim & Overpass)

## 📊 Machine Learning Modules
The system utilizes ensemble-ready clinical parameters to train predictive models:
1.  **Diabetes**: Analyzes FBG, HbA1c, RBS, Blood Pressure, and BMI.
2.  **Heart Disease**: Evaluates Age, BP, Cholesterol (HDL/LDL/Triglycerides), and lifestyle factors like smoking and exercise.
3.  **Kidney Disease**: Focuses on Blood Urea, Serum Creatinine, Hemoglobin, and Specific Gravity.
4.  **Liver Disease**: Monitors Bilirubin levels, Enzymes (SGPT/SGOT), and Protein ratios.
5.  **Breast Cancer**: Classifies tumors based on mean radius, texture, perimeter, and area.

## 💻 Installation & Setup

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/yourusername/medalert-ai.git
    cd medalert-ai
    ```

2.  **Install Dependencies**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Environment Configuration**
    Create a `.env` file in the root directory and add your credentials:
    ```env
    SECRET_KEY=your_flask_secret_key
    MONGO_URI=your_mongodb_connection_string
    GROQ_API_KEY=your_groq_api_key
    ```

4.  **Train the Models**
    Ensure your datasets are in the `datasets/` folder and run the training script:
    ```bash
    python train_models.py
    ```

5.  **Run the Application**
    ```bash
    python app.py
    ```
    The application will be available at `http://127.0.0.1:5000`.
